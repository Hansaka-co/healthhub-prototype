# HealthHub — High-Fidelity Prototype (CW2)

A mobile health-admin app prototype supporting **SDG 3: Good Health and Well-being**.
Main task: add a health appointment, set reminders, and confirm it on the dashboard.

Built with **HTML, CSS and JavaScript** (no frameworks, no templates) for the
5COSC025W Human Computer Interaction & User Experience module.

## Run it
Open `index.html` in any web browser — no installation needed.

## Files
- `index.html` — page structure and screens
- `styles.css` — visual design system (colours, typography, components)
- `app.js` — navigation, calendar, reminders, and app logic
